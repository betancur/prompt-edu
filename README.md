# Prompt Edu 🎓

Prompt Edu es una plataforma educativa diseñada para empoderar a los educadores en la era de la Inteligencia Artificial. Nuestra misión es proporcionar herramientas, recursos y conocimientos que permitan a los profesores integrar efectivamente la IA en sus prácticas pedagógicas.

## 🌟 Características Principales

- **Biblioteca de Prompts**: Colección curada de prompts educativos organizados por materias y objetivos pedagógicos.
- **Centro de Recursos**: Acceso a guías, documentos, videos y herramientas prácticas para la implementación de IA en el aula.
- **Comunidad Educativa**: Espacio para compartir experiencias y mejores prácticas con otros educadores.
- **Herramientas Prácticas**: Suite de utilidades diseñadas específicamente para tareas educativas con IA.

## 🚀 Tecnologías

- React.js
- Tailwind CSS
- Shadcn/ui
- Next.js
- Node.js

## 📋 Requisitos Previos

- Node.js (versión 18 o superior)
- npm o yarn
- Git

## 🛠️ Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/prompt-edu.git
```

2. Navega al directorio del proyecto:
```bash
cd prompt-edu
```

3. Instala las dependencias:
```bash
npm install
# o
yarn install
```

4. Inicia el servidor de desarrollo:
```bash
npm run dev
# o
yarn dev
```

5. Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver la aplicación.
